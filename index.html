Common help topics

These are the most common questions for users and App Store reviewers.

Activation QR code

Use the camera to scan the activation QR code. The app uses camera access only for activation scanning.

Bluetooth reader connection

The app uses Bluetooth to discover, connect to, and exchange data with nearby RFID readers.

NFC card operations

NFC is used to read from and write to supported RFID tags during card operations.

Custom background image

The photo library is used only so users can select a custom background image.

FAQ

Answers written for the current app flow.

Why does the app need camera access?

The camera is used only to scan activation QR codes.

Why does the app need Bluetooth?

Bluetooth is used to discover, connect to, and exchange data with nearby RFID readers.

Why does the app need NFC?

NFC is required for supported card operations. The app reads from and writes to compatible RFID tags when the hardware supports it.

Why does the app ask for photo library access?

Photo library access is used only to let users choose a custom background image.

What should I do if Bluetooth or NFC does not work?

Make sure Bluetooth is turned on, the RFID reader is nearby and powered, and the device supports NFC if you are using NFC features. If the issue continues, email support with a screenshot and short description.
